# 🏛️ UrbanEye — AI-Powered Smart City Governance Platform

UrbanEye is a modern Smart City Governance Platform designed to streamline citizen grievance reporting, administrative monitoring, geospatial complaint tracking, and AI-assisted governance insights.

The platform enables citizens to report civic issues while providing administrators with powerful tools for complaint management, analytics, hotspot detection, and city-wide monitoring.

### 🤖 NAGARI — Smart City Intelligence Assistant

NAGARI (National Assistance for Governance & Administrative Response Intelligence) acts as the platform's governance assistant.

Capabilities include:

- Complaint Analysis
- Priority Detection
- Department Recommendations
- Hotspot Awareness
- Administrative Insights
- Complaint Statistics Queries
